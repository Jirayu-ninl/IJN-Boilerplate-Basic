## TheIceJI-Boilerplate (Basic)
**update Feb 16, 2022**

**NEXT JS** with custom config webpack and Babel

#### Features

- TypeScript support
- Next Config (NextOffline, Path alias)
- Header Config (SEO, Google Tag, Font)
- EmotionCSS and TailwindCSS
- Global CSS
- Global Config

#### Structure

````
root/
├─ app/
│  ├─ config/
├─ pages/
├─ public/
├─ resources/
│  ├─ content/
│  │  ├─ global/
│  ├─ views/
│  │  ├─ theme/
│  │  ├─ pages/
│  │  │  ├─ (each page)/
│  │  │  │  ├─ components/
│  │  │  │  ├─ styles/
│  │  │  │  ├─ assets/
````

#### How to run

1. Add you config files at `/app/config`
2. Install
````
yarn
````
1. Run
````
yarn dev
````
